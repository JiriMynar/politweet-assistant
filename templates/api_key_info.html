Pro správnou funkci aplikace Factchecker Asistent je nutné mít platný OpenAI API klíč. Tento klíč můžete získat na stránkách OpenAI.

Možnosti nastavení API klíče

1. Pomocí proměnné prostředí

Nastavte proměnnou prostředí OPENAI_API_KEY s hodnotou vašeho API klíče.

# Pro Linux/Mac
export OPENAI_API_KEY=váš_api_klíč

# Pro Windows
set OPENAI_API_KEY=váš_api_klíč

2. Pomocí konfiguračního souboru

Vytvořte soubor config.json v kořenovém adresáři aplikace s následujícím obsahem:

{
  "openai_api_key": "váš_api_klíč"
}

Bezpečnostní doporučení

  • Nikdy nesdílejte svůj API klíč s ostatními.
  • Neukládejte API klíč do veřejných repozitářů.
  • Pravidelně kontrolujte využití vašeho API klíče na stránkách OpenAI.
  • V případě podezření na zneužití klíče jej okamžitě zneplatněte a vytvořte nový.
